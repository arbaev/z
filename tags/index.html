indesign

Шпаргалка по InDesign

Концепция “объект и кадр”.\ Объект может быть любой формы.\

ctrl-D - импорт картинок, одной или нескольких. вставка происходит по очереди, смена очереди - стрелки лево-право. Если зажать ctrl, то вставятся сразу все картинки сеткой

V - черная стрелка - двигает объект
A - белая стрелка - двигает кадр

A-Master - общая подложка под все страницы, нижний слой

Library - сохранять свои темплейты, перед этим удалив с них фото

Включить в настройках high performance для качественных превьюшек фото.

w - быстрый предпросмотр shift-w - полный предпросмотр

ctrl-2 (3) - приблизить к выбранному объекту,
ctrl-0 - масштаб страницы на экран,
ctrl-alt-0 - разворот на экран

ctrl-[, ctrl-] - слой вверх/вниз

ctrl-e - пропорционально вписать картинку в объект (обрезка)
ctrl-shift-e - уменьшить картинку по размерам объекта (поля)
ctrl-shift-c - центрировать картинку в объекте
alt-e - изменить размер объекта по размеру картинки

ctrl-alt-shift-v - вставить скопированную картинку в объект

ctrl-b — свойства объекта, в том числе паддинги
